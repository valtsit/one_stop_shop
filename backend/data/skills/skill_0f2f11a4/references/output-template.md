# 输出模板

> 面试题输出的格式规范和示例。

---

## 输出格式

每次输出包含三个部分：

### Part 1: 简历分析摘要

简要说明候选人画像、关键事件、风险点和建议考察维度。

### Part 2: 面试题（核心输出）

按考察维度分类，每个维度包含 2-4 道题。

### Part 3: 追问建议

针对简历中的关键经历，给出追问方向。

---

## 单题格式

每个面试题按以下格式输出：

```
### [维度名] - [子维度名]

**题目 N：** [问题内容]

- **考察点：** 这道题考察的是什么（一句话）
- **评估标准：** 好的回答应该包含哪些要素
- **追问方向：**
  - 如果候选人回答A方向 → 追问 [具体内容]
  - 如果候选人回答B方向 → 追问 [具体内容]
- **简历关联：** 这道题与简历中 [具体经历/风险点] 的关联
```

---

## 完整输出示例

假设岗位为"运营专员"，候选人有3年电商运营经验但最近一份工作不到1年离职。

---

## 一、简历分析摘要

**候选人画像：**
- 工作年限：3年
- 行业背景：电商 → 电商 → 电商
- 岗位变迁：运营助理 → 运营专员 → 运营专员
- 当前状态：离职

**关键事件：**
1. 在第二家公司负责某品类店铺运营，月销从5万提升至20万
2. 第三份工作仅8个月即离职

**风险点：**
- 最近一份工作不足1年离职 — 需考察稳定性（基本面）
- 两次同级别无晋升 — 需考察成长驱动力（底层逻辑）

**亮点：**
- 有明确的业绩提升数据 — 可深挖执行力和方法论

**建议考察维度（按优先级）：**
1. 基本面（稳定性）— 最近离职原因
2. 行动力（执行力）— 有业绩数据可深挖
3. 愿力（行业认同）— 三年同行经验
4. 岗位职责相关 — 专业技能

---

## 二、面试题

### 维度1：基本面（稳定性）

**题目 1：** 你在上一家公司只待了8个月就离开了，方便说一下原因吗？

- **考察点：** 离职原因的真实性和合理性
- **评估标准：** 好的回答应客观描述原因，不过度抱怨前公司，且离开原因与本次求职动机有逻辑关联
- **追问方向：**
  - 如果说"公司原因" → 追问："当时有没有尝试过沟通或改变现状？做了什么？"
  - 如果说"个人发展" → 追问："那你当时入职时看重的点是什么？什么时候发现不合适了？"
  - 如果说"客观原因（搬迁/裁员等）" → 追问："如果这个客观因素没有发生，你还会继续做下去吗？为什么？"
- **简历关联：** 第三份工作仅8个月离职

**题目 2：** 一般来说，什么情况下你会考虑换工作？

- **考察点：** 候选人的职业稳定性和决策逻辑
- **评估标准：** 有清晰的换工作标准（如成长空间、文化匹配），而非冲动决定
- **追问方向：**
  - 如果回答模糊 → 追问："你过往换工作的经历中，哪一次是最纠结的？纠结了多久？最后怎么决定的？"
- **简历关联：** 三年换了三份工作

---

### 维度2：行动力（执行力）

**题目 3：** 你提到在第二家公司把某品类月销从5万做到20万，能详细说说你具体做了哪些事情吗？

- **考察点：** 业绩背后的真实贡献和个人方法论
- **评估标准：** 能清晰描述具体动作（而非只说"优化了运营"），有因果逻辑，能区分自己的贡献和团队/平台的作用
- **追问方向：**
  - 如果描述笼统 → 追问："这15万的增长，你认为最关键的一步是什么？为什么？"
  - 如果说得很详细 → 追问："过程中遇到的最大阻碍是什么？你怎么解决的？"
  - 如果功劳全归自己 → 追问："团队其他人在这个过程中分别做了什么？"
- **简历关联：** 简历中的业绩数据

---

### 维度3：愿力（行业认同）

**题目 4：** 你做了3年电商运营，是什么让你一直选择这个行业？

- **考察点：** 对行业的热情和长期规划
- **评估标准：** 有真实的行业兴趣（而非只是"碰巧做了这行"），对未来在行业内的发展方向有思考
- **追问方向：**
  - 如果说"碰巧" → 追问："做了3年之后，你现在对电商行业有什么自己的理解？觉得未来的机会在哪里？"
- **简历关联：** 三年电商行业经验

---

### 维度4：岗位职责相关

**题目 5：** 你觉得一个运营专员最重要的三项能力是什么？你觉得自己最擅长哪一项？

- **考察点：** 对岗位的理解深度和自我认知
- **评估标准：** 对运营岗位有务实的理解（不是背概念），自我评估与简历经历能对得上
- **追问方向：**
  - 回答后 → 追问："能举一个你发挥这项能力的具体例子吗？"
  - 如果回答很理论 → 追问："你说的这些能力里，哪一项是你实际工作中用得最多的？"
- **简历关联：** 运营岗位匹配度

---

## 三、追问建议

针对简历中的关键点，以下是补充追问方向：

**针对"月销5万→20万"的经历：**
- 用了多长时间达到20万？
- 当时的预算是多少？ROI怎么样？
- 如果让你再做一次，你会有什么不同做法？

**针对"3年3份工作"的经历：**
- 三份工作中，哪份让你最有成就感？为什么？
- 你觉得自己在职业选择上有没有可以改进的地方？
