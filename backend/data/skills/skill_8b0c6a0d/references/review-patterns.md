# 评语风格模式与真实示范

## 真实买家的说话特征

### 真实买家会这样写

- **直接说感受**："片大，干净，不错。" —— 没有铺垫，没有"我买了这个因为..."
- **碎片化**：想到什么说什么，不追求完整
- **用词朴素**："还行"、"挺好"、"不错" —— 不会每条都用"非常满意"
- **有瑕疵感**：偶尔提点小问题（"物流慢了点"、"有点贵但值得"）
- **不会用排比**：不会"首先...其次...最后..."
- **省略句多**："料挺足的"、"包装可以"、"发货快"
- **语气词多**："还行吧"、"挺好的"、"不错不错"

### 真实买家不会这样写

- ❌ "作为一个注重养生的人，我一直在寻找高品质的黄芪，经过多方对比，最终选择了这家店..."
- ❌ "因为最近气色不好，听朋友说这个养生茶效果不错，所以买来试试，没想到真的很有效果..."
- ❌ "首先包装很好，其次品质不错，最后物流也很快，总之非常满意！"
- ❌ "选用道地药材，品质上乘，值得推荐。"

## 卖点融入正反对比

### 原料类

| 卖点 | ❌ 生硬写法 | ✅ 自然写法 |
|------|------------|------------|
| 片大 | 片大饱满，品相优良 | 片大，厚实 |
| 干净 | 干净无杂质，品质保证 | 没什么碎渣 |
| 药香 | 药香浓郁，道地正宗 | 药香浓，闻着就正 |
| 原产地 | 岷县发货，道地产区 | 岷县的，放心 |
| 不熏硫 | 无硫熏蒸，天然健康 | 没熏过，闻着自然 |
| 回购 | 多次回购，品质稳定，值得信赖 | 第三次买了，一直很稳 |

### 养生茶类

| 卖点 | ❌ 生硬写法 | ✅ 自然写法 |
|------|------------|------------|
| 口感 | 口感醇厚，回味甘甜 | 味道还行，能坚持 |
| 配料 | 配料齐全，科学配比 | 配料挺全的，都是整颗 |
| 方便 | 独立包装，方便携带 | 一天一包，省事 |
| 功效 | 具有安神助眠的功效 | 喝了几天，感觉还行 |
| 原料 | 精选优质原料，品质上乘 | 料挺足的，没有碎渣 |

### 礼盒类

| 卖点 | ❌ 生硬写法 | ✅ 自然写法 |
|------|------------|------------|
| 档次 | 包装精美，高端大气 | 包装可以，挺有档次 |
| 价值 | 性价比高，物有所值 | 100多看着像300的，值了 |
| 送礼 | 送礼佳品，体面大方 | 送长辈的，拿得出手 |
| 品质 | 里头药材品质优良，不是样子货 | 里头药材也不错，不是样子货 |
| 反馈 | 送礼反馈良好，对方非常满意 | 送出去了，长辈很喜欢 |

## 评语长度示范

### 短评（5-15字）

**原料类**：
- "片大，均匀，不错。"
- "药香浓，正宗。"
- "干净，没碎渣。"
- "回购的，品质稳定。"
- "岷县货，放心。"
- "闻着就正。"
- "料挺足。"

**养生茶类**：
- "口感还行，能坚持。"
- "配料齐全，方便。"
- "一天一包，省事。"
- "喝着挺舒服。"
- "料很足。"
- "味道可以。"

**礼盒类**：
- "包装有档次。"
- "送人合适。"
- "值了。"
- "礼盒精致。"
- "发货快。"
- "拿得出手。"

### 中评（15-40字）

**原料类**：
- "片大均匀，没什么碎渣，煮水味道正。"
- "第三次买了，品质一直稳定，岷县发货的。"
- "闻着药香很浓，没有异味，干净。"
- "比我在药店买的好，还便宜。"
- "颗粒饱满，个头挺大，颜色也正。"
- "打开盖子药香扑鼻，片大厚实。"

**养生茶类**：
- "喝了一周，感觉还行，口感不难喝。"
- "一天一包，方便，配料挺全的。"
- "喝了半个月，气色好像好了一些。"
- "配料都是整颗的，没有碎渣，料很足。"
- "比想象中好喝，能坚持，独立包装方便。"
- "味道可以，不难喝，早上泡一包能喝一天。"

**礼盒类**：
- "礼盒精致，比图片好看，送人拿得出手。"
- "这个价格这个品质，值了，药材也不错。"
- "送领导的，很有面子，发货也快。"
- "100多块钱，看着像300多的，划算。"
- "里头药材品质也不错，不是样子货。"
- "包装严实，礼盒完好，送人放心。"

### 略长（40-70字）

**原料类**：
- "打开盖子药香扑鼻，片大厚实，没有碎渣。煮了几次水，味道很足，耐泡。这个价格买到这个品质，值了。"
- "第三次回购了，品质一直稳定。岷县发货的，正宗。片大均匀，干净，煮水药香浓。比药店便宜。"
- "收到就打开了，片大干净，没什么碎渣。药香很浓，闻着就正。岷县的货，放心。煮了几次水，味道很足。"

**养生茶类**：
- "喝了两周，感觉睡眠好了一点，口感微甜能坚持。一天一包，独立包装方便。配料都是整颗的，没有碎渣，料很足。"
- "坚持喝了一个月，感觉比之前精神点。味道可以，不难喝，早上泡一包能喝一天。配料齐全，品质不错。"
- "喝了一周多，口感还行，能坚持。配料挺全的，都是整颗的，没有碎渣。一天一包，独立包装方便。"

**礼盒类**：
- "礼盒比想象中好，很有质感。里头药材品质也不错，不是样子货。送长辈的，很满意，100多块钱看着像300多的。"
- "送领导的，很有面子。包装严实，礼盒完好。里头药材品质很好，不是只看包装的样子货。发货也快。"
- "包装可以，挺有档次。里头药材也不错，不是样子货。送长辈的，拿得出手。100多块钱，值了。"

## 主评 vs 追评对比示范

### 原料类

**主评**："片大厚实，药香浓，岷县发货的，正宗。"
**追评（7天后）**："用了一周多，品质稳定，药香一直很浓。准备再买一点。"

**主评**："第三次买了，干净没碎渣，品质一直稳定。"
**追评（10天后）**："回购的，这次搭配枸杞一起泡，味道更好。认准这家了。"

**主评**："片大，干净，药香正。"
**追评（12天后）**："用了十来天，品质很稳。比药店便宜，认准这家了。"

### 养生茶类

**主评**："配料挺全的，都是整颗的，独立包装方便。"
**追评（5天后）**："喝了几天，口感还行，能坚持。一天一包，省事。"

**主评**："一天一包，方便，原料看着不错。"
**追评（12天后）**："喝了十来天，感觉气色好像好了一些。口感微甜，好入口，会继续喝。"

**主评**："味道可以，不难喝，配料挺全的。"
**追评（8天后）**："喝了一周多，感觉还行吧，能坚持。一天一包，方便。"

### 礼盒类

**主评**："包装很有档次，送长辈的，很满意。"
**追评（3天后）**："送出去了，长辈很喜欢，说品质不错。礼盒精致，拿得出手。"

**主评**："100多块钱，看着像300多的，值了。里头药材品质也不错。"
**追评（8天后）**："朋友收到很满意，说很有档次。下次送礼还来买。"

**主评**："包装可以，挺有档次。送领导的，拿得出手。"
**追评（5天后）**："送出去了，领导说不错。礼盒精致，有面子。"

## 追评时间分布参考

追评时间不要统一，要错开：

- 3天后：适合礼盒类（刚送出去就有反馈）
- 5天后：适合养生茶类（刚喝几天，初步感受）
- 7天后：通用（用了一周，有基本判断）
- 10天后：适合养生茶类（效果开始显现）
- 12-15天后：适合原料类（用了一段时间，品质确认）

每次生成的追评时间要随机选择，不要有规律。

## 常见错误与修正

| 错误写法 | 问题 | 修正后 |
|----------|------|--------|
| "因为最近睡眠不好，所以买了这个茶，喝了几天感觉效果不错。" | 编故事、因果叙事 | "喝了几天，感觉还行，会坚持喝。" |
| "作为一个养生爱好者，对比了很多家，最终选择了这家，果然没让我失望。" | 刻意铺垫、身份设定 | "买过几家，这家最满意。" |
| "首先包装很好，其次品质不错，最后物流快，总之非常满意。" | 结构化表达、模板感 | "包装严实，品质不错，发货也快。" |
| "喝了一天，睡眠完全好了，再也不失眠了。" | 短期夸大、绝对化 | "喝了几天，感觉挺舒服的。" |
| "这个茶效果神奇，立竿见影，强烈推荐。" | 夸大、禁用词 | "口感还行，能坚持喝。" |
| "选用道地药材，品质上乘，值得推荐。" | 广告语、书面语 | "岷县的，品质不错。" |
| "品质优良，物有所值，性价比很高。" | 书面语堆砌 | "品质不错，值了。" |
| "具有补气养血、健脾益肺的功效。" | 广告语、功效堆砌 | "喝着挺舒服的。" |
