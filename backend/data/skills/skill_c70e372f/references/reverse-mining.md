# 反向挖掘方法论

不知道同行怎么做，就不知道什么算差异化。冲突感来自正反对比。

---

## 核心理念

**反向挖掘**不是简单的"找不同"，而是：

1. **先了解行业默认操作** → 知道什么是"大家都在做的"
2. **再推导稀缺事实** → 知道什么是"很少有人做的"
3. **最后构建感知差距** → 让消费者自己感受到差距

---

## 挖掘三步法

### 第一步：揭露行业潜规则

**目的**：让消费者知道"原来大家都在这样做"。

**提问方式**：
- "这个维度上，同行通常是怎么做的？"
- "有哪些'行业默认操作'是消费者不知道的？"
- "有哪些做法是'看起来好，实际一般'的？"

**示例**（以"产地加工与炮制"维度为例）：
- 行业默认：用电蒸设备快速蒸制，效率高、成本低
- 消费者不知道：电蒸温度均匀但缺少"火候变化"，影响药性转化
- 看起来好：标注"九蒸九晒"，实际可能是机器蒸+烘干

### 第二步：推导稀缺事实

**目的**：找到"同行很少做/不愿做/做不到"的点。

**提问方式**：
- "你的产品在这个维度上，有什么是同行很少做的？"
- "这个优势背后的'代价'是什么？"
- "同行为什么做不到/不愿做？"

**示例**：
- 稀缺事实：坚持柴火慢蒸，每锅蒸制时间是电蒸的3-4倍
- 背后代价：产量低、人工成本高、无法标准化
- 同行不愿做：效率太低，利润空间被压缩

### 第三步：构建感知差距

**目的**：用一句话让消费者感知到差距，而不是直接说"我们更好"。

**表达公式**：
- **代价暗示法**："电蒸是效率，柴火慢蒸是诚意"
- **时间成本法**："这份等待，吃得出"
- **选择诚意法**："不是做不到，是不愿将就"
- **体感验证法**："颜色能骗人，身体骗不了"

---

## 感知差距的表达模式

### 模式1：代价暗示

**公式**：[行业常见做法] 是 [效率/成本]，[我们的做法] 是 [诚意/品质]

**示例**：
- "机器切片是效率，手工切片是诚意"
- "一年采收是产量，三年等待是品质"
- "批量烘干是成本，自然晾晒是讲究"

### 模式2：时间成本

**公式**：[时间代价]，[体感结果]

**示例**：
- "这份等待，吃得出"
- "8年休耕，只为这一口"
- "慢工出细活，急不得"

### 模式3：选择诚意

**公式**：不是 [做不到]，是 [不愿将就]

**示例**：
- "不是做不到快，是不愿将就"
- "不是没有更便宜的方法，是不愿降低标准"
- "不是不能批量，是不愿丢掉手工的温度"

### 模式4：体感验证

**公式**：[消费者可感知的验证方式] 才是 [真正的品质]

**示例**：
- "颜色能骗人，身体骗不了"
- "手掰拉丝，入口软糯，才是真的九制到位"
- "不口干、不上火，才是真的蒸透了"

## 挖掘输出格式

对每个勾选维度，输出以下结构：

```
维度：[维度名称]

行业潜规则：
- [同行通常做法1]
- [同行通常做法2]
- [消费者不知道的真相]

稀缺事实：
- [用户产品的独特之处]
- [背后的时间/成本/工艺代价]
- [同行为什么做不到/不愿做]

感知差距：
- [一句话表达，使用上述表达模式之一]
```

---

## 注意事项

1. **不要编造**：稀缺事实必须基于用户提供的真实信息
2. **不要贬低**：揭露潜规则不等于说竞品不好，而是让消费者知道"原来可以这样做"
3. **不要空泛**：感知差距必须具体到可感知的体感或代价
4. **保留冲突感**：正反对比才有冲击力，不要和稀泥
