# 模板库与品牌理念融入示例

## 11.1 原料类·内行朋友模式（V11-V14五维度分层）

适用：当归、黄芪、党参、枸杞、甘草、牛蒡根等原药材。

**原料线采用五维度分层，模板如下：**

| 维度 | 层级名称 | 示例 |
| :--- | :--- | :--- |
| **产品名称层**（第1层，最重要） | 产品名称/基础属性 | "山野老芪""深山当归""沙地甜蒡" |
| **买点层**（第2层） | 顾客买点（4+4） | "断面分明，道地可鉴""泡水煲汤，补气养元""菊花心纹，肉质饱满" |
| **差异层**（第3层） | 差异化卖点（4+4） | "岷县原产，海拔两千""甘肃直发，产区新切""无硫晾晒，自然本色" |
| **利益层**（第4层） | 价格利益点（4+4） | "源头直供，省去中间""两罐共斤，足吃两月""比价药店，足足省半" |
| **保障层**（第5层） | 品质保障/品牌背书 | "PICC承保""7天退""假一赔""附检测报告" |

关键词：道地、足干、无硫、菊花心、海拔、药香、产地直供、主根大片、产区新切

**画面感命名思路库（V14）：**

**以下不是"填空词库"，而是帮助你思考产品真实属性的思路框架。命名必须基于产品实际信息，从以下角度挖掘真实素材：**

| 挖掘角度 | 问自己什么问题 | 好词的方向（有画面/有故事） | 差词的方向（无画面/像标签） |
| :--- | :--- | :--- | :--- |
| **生长环境** | 产品在哪里生长？什么自然环境？ | 山野（看到山林）、深山（看到深山）、林下（看到树荫） | 苍山（只是一个县名）、山东（只是一个省名） |
| **种植方式** | 产品怎么种的？土壤有什么特点？ | 沙地（看到沙土）、荒野（看到荒凉） | 沙培（技术参数）、有机（认证标签） |
| **年份/时间** | 产品长了多久？有什么时间积累？ | 老（经历年份）、陈（经过存放）、古（有历史感） | 三年（只是一个数字）、陈化（加工术语） |
| **口感/感官** | 产品吃起来/看起来什么感觉？ | 甜（尝到甜味）、香（闻到香味）、金（看到金黄） | 菊糖含量高（检测指标）、色泽好（空泛描述） |
| **产地意象** | 产地本身能唤起什么联想？ | 岷山（唤起道地感）、草原（看到广阔） | 兰陵（只是一个县名）、江苏（只是一个省名） |

**命名验证标准（每个命名必须通过）：**
1. **画面检验**：读出这个命名，脑中是否能浮现一个画面？
2. **故事检验**：这个命名是否暗示了一段经历或一种状态？
3. **力量检验**：每个字是否都有信息量，没有凑数的？
4. **事实检验**：命名中每个词是否都是产品的真实属性？

**满分案例拆解（作为校准标准）：**

| 命名 | 画面感 | 故事感 | 力量感 | 事实基础 |
| :--- | :--- | :--- | :--- | :--- |
| 山野老芪 | "山野"→看到一片野生山林 | "老"→经历了年份生长 | 每个字都沉 | 山野生长+老黄芪（真实） |
| 深山当归 | "深山"→看到原始深山环境 | 暗示"深山里长出来的" | 4字干净 | 深山环境（真实） |
| 沙地甜蒡 | "沙地"→看到沙质土地 | "甜"→口感特征像在说"尝起来甜" | 两个词都有感知价值 | 沙地种植+甜味（真实） |

## 11.2 养生茶·懂你的朋友模式

适用：陈皮茯苓茶、酸枣仁百合茶、祛湿茶、姜枣茶等组合茶饮。

- 主标题（8字，4+4）："湿去轻盈，气色自现""晨起有光，夜得深眠""古方温养，润物无声"
- 辅标题1（8字，4+4）："陈皮三年，理气为先""十五味方，古法配比""真材实料，三泡犹香"
- 辅标题2（8字，4+4）："买二送一，足喝一月""半价尝新，囤着慢喝""日均几角，随时一杯"
- 次要内容1（4-8字）："陈皮茯苓茶/30包""祛湿茶/一个月量"
- 次要内容2（2-4字）："喝不惯退""老客复购""不满意退"

关键词：古方配比、三年陈皮、理气、轻盈、温养、料足耐泡

## 11.3 礼盒·送礼参谋模式

适用：当归礼盒、黑枸杞礼盒、滋补礼盒、年货礼盒等。

- 主标题（8字，4+4）："以礼传心，贵在知人""开盒见诚，心意自重""珍重之人，不负此礼"
- 辅标题1（8字，4+4）："颗颗溯源，精选大果""道地原材，检测随附""人工甄选，品质可鉴"
- 辅标题2（8字，4+4）："好礼不贵，体面到位""附赠手袋，拎着即走""开盒即送，省心省力"
- 次要内容1（4-8字）："当归尊享礼盒200g""黑枸杞礼盒240g"
- 次要内容2（2-4字）："破损包赔""拆封可退""顺丰直达"

关键词：体面、知人、见诚、有底气、不落俗套

## 11.4 品牌理念融入示例

以下示例展示文案如何通过**方向、气质、价值观**体现品牌理念。文案中**不需要出现**品牌理念词，也**不需要刻意用其他词汇替代**。

### 原料类·体现专注本源精神（当归·五维度分层）

- 产品名称层："深山当归"——画面感（深山=看到原始深山环境）+品类（当归），4字干净，有画面、有力量
- 买点层："断面分明，道地可鉴"——专注打一个鉴别卖点，不堆砌，体现"专注本源"的精神
- 差异层："岷县原产，海拔两千"——只给一个产地支撑，不贪心，体现"纯正不杂"的精神
- 利益层："源头直供，省去中间"——价格利益清晰直接，不绕弯
- 保障层："PICC承保"——具体可兑现的信任信号
- **理念解读**：这组文案没有品牌理念词，但"只打一个卖点+只给一个支撑"的方向体现了专注本源的精神。产品名称层用"深山当归"替代"野生当归"，有场景画面感，拒绝市面通用名，体现品牌差异。

### 养生茶·体现温和平衡精神（陈皮茯苓茶）

- 主标题："湿去轻盈，气色自现"——平和描述好状态，不恐吓、不制造焦虑，体现"温和平衡"的精神
- 辅标题1："陈皮三年，理气为先"——用配方讲究建立信任，不夸大单一功效
- 辅标题2："买二送一，足喝一月"——价格利益直接明了
- 次要内容1："陈皮茯苓茶/30包"
- 次要内容2："喝不惯退"
- **理念解读**：这组文案没有品牌理念词，但"平和描述+不夸大功效"的调性体现了温和平衡的精神。

### 礼盒·体现品质底气精神（当归礼盒）

- 主标题："以礼传心，贵在知人"——有分量、有底气，不流俗，体现"底气自足"的精神
- 辅标题1："颗颗溯源，精选大果"——用真实品质建立信心，不编造
- 辅标题2："好礼不贵，体面到位"——价格与体面平衡，不唯便宜论
- 次要内容1："当归尊享礼盒200g"
- 次要内容2："破损包赔"
- **理念解读**：这组文案没有品牌理念词，但"送的是健康生活方式+品质有底气"的方向体现了品质底气精神。

**融入要点：**
- 品牌理念体现在**文案方向**上（选题角度、语气调性、价值观），不是词汇上
- 原料类：专注打透一个卖点 = 专注本源
- 养生茶：温和不夸大 = 温和平衡
- 礼盒：品质底气+生活方式 = 品质自信
- 不要为了让文案"看起来有理念"而修改原本好的词汇
