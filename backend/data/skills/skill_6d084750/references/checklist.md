# 完整自检清单

生成最终答案前，必须逐项检查：

## 品牌调研检查

- [ ] 是否执行了4.0A品牌信息收集步骤？用户是否有品牌？
- [ ] **有品牌时**：是否执行了4.0品牌风格前置调研？
- [ ] **有品牌时**：品牌调研是否从相关维度（定位/视觉/产品线/客群/文案风格/差异化/品牌理念如适用）逐条分析？
- [ ] **有品牌时**：搜索结果是否逐条引用了实际内容，未编造品牌信息？
- [ ] **有品牌时**：搜索失败时是否使用了固定话术？
- [ ] **无品牌时**：是否已跳过4.0品牌调研，标注"无品牌信息，以品类通用调性为准"？
- [ ] 文案调性是否与品牌调研结果一致（有品牌）/ 是否与品类通用调性一致（无品牌）？
- [ ] 输出格式中是否标注了"品牌调性"和"品牌理念融入"两行？
- [ ] 文案解读中是否包含"品牌调性适配说明"和"品牌理念融入说明"？

## 交互流程检查

- [ ] 是否联网搜索了同品类竞品主图文案？
- [ ] 搜索结果是否逐条引用了实际内容，未编造竞品信息？
- [ ] 搜索失败时是否使用了固定话术？
- [ ] 用户表示不确定卖点方向时，是否推荐了竞品图分析或差评数据分析路径？
- [ ] 竞品图分析是否逐张拆解、汇总共性短板、提出差异化方向？
- [ ] 差评分析是否结构化分类痛点、明确对应卖点方向？
- [ ] 是否逐个问了4个问题，每问前都联网给参考？
- [ ] 问题1答案是否满足充分性标准（至少一个具体卖点方向）？不满足时是否追问了？
- [ ] 问题2答案是否满足充分性标准（至少一个具体差异点）？不满足时是否追问了？
- [ ] 问题3答案是否满足充分性标准（明确自用/送礼，送礼有场景）？不满足时是否追问了？
- [ ] 问题4答案是否满足充分性标准（有方向或明确说"你帮我定"）？不满足时是否追问了？
- [ ] 输出文案前是否进行了充分性终审？
- [ ] 所有过渡语、提问语是否使用了固定话术？

## 文案质量检查（养生茶/礼盒）

- [ ] 产品线是否正确识别？语感模式是否锁定？
- [ ] 五层文案是否齐全？每层是否有明确内容？
- [ ] 主标题是否严格8字（4+4）？是否有分量感？
- [ ] 辅标题1是否严格8字（4+4）？是否有至少一个**具体的、可验证的专业细节**？
- [ ] 辅标题2是否严格8字（4+4）？是否只谈入手理由？
- [ ] 次要内容1是否准确无歧义？
- [ ] 次要内容2是否有**具体、可兑现**的保障承诺（不是空话）？
- [ ] 整体文案是否**读着有分量、有节奏**？（4+4对仗是否工整）
- [ ] 排重自查：各层角度不同，但能形成"完整推销"的逻辑链？
- [ ] 是否有违禁词（对照全局违禁词库）？
- [ ] 主图总字数是否在28-36字范围？
- [ ] 是否附了"文案解读"？
- [ ] 风格引擎调性是否一致，未跨风格混杂？

## 文案质量检查（原料线·五维度分层）

- [ ] 产品线是否正确识别为"原料类"？
- [ ] 五维度是否齐全？每层是否有明确内容？
- [ ] 产品名称层是否4-6字？是否读着有画面感、故事感、力量感？每个字都有信息量？
- [ ] **产品名称层是否采用事实驱动画面感命名，拒绝市面通用等级词（野生/特级/精选/优级/一级）和标签式/参数式命名？是否通过画面感、故事感、力量感三项检验？**
- [ ] 买点层是否严格8字（4+4）？是否从顾客视角出发？
- [ ] 差异层是否严格8字（4+4）？是否有至少一个**具体的、可验证的专业细节**？
- [ ] 利益层是否严格8字（4+4）？是否只谈入手理由？
- [ ] 保障层是否有**具体、可兑现**的保障承诺（不是空话）？
- [ ] 是否进行了五维度打分（1-5分）？每项是否给了评分理由？
- [ ] 总分是否在23分以上？单项是否均不低于3分？
- [ ] 整体文案是否**读着有分量、有节奏**？（4+4对仗是否工整）
- [ ] 排重自查：各层角度不同，但能形成"完整推销"的逻辑链？
- [ ] 是否有违禁词（对照全局违禁词库）？
- [ ] 主图总字数是否在28-36字范围？
- [ ] 是否附了"文案解读"（含五维度评分解读）？

## 品牌理念融入检查

- [ ] 是否执行了4.0A品牌信息收集步骤？用户是否有品牌？
- [ ] **有品牌+有理念**：文案方向是否体现理念精神？
- [ ] **有品牌+有理念**：品牌理念融入程度是否与用户描述一致？
- [ ] **有品牌+有理念**：文案中是否**未刻意植入**理念关键词？（理念通过方向体现，不通过词汇植入）
- [ ] **有品牌+无理念** / **无品牌**：文案是否以产品卖点为核心，未硬套理念框架？
- [ ] **养生茶/礼盒**：主标题是否以顾客买点为前提？（不是纯理念口号）
- [ ] **原料线**：买点层是否以顾客收益为前提？（不是纯理念口号）
- [ ] **养生茶/礼盒**：辅标题1是否有具体支撑（产地/工艺/配方/数据）？
- [ ] **原料线**：差异层是否有具体支撑（产地/工艺/配方/数据）？
- [ ] 文案解读中是否包含"品牌理念融入说明"？（有品牌+有理念：说明如何体现；无品牌/无理念：说明以产品卖点为核心）

## 防幻觉检查

- [ ] 所有引用的竞品信息是否来自实际搜索结果？
- [ ] 是否有编造的竞品名称、品牌名称、店铺名称？
- [ ] 搜索失败时是否如实告知用户？
- [ ] 不确定的信息是否使用了"我不确定"的固定话术？
- [ ] 竞品图分析是否基于图片实际可见内容，未推测图中没有的信息？
- [ ] 差评分析是否基于用户实际提供的数据，未自行编造差评内容？
- [ ] 品牌调研是否基于实际搜索结果，未编造品牌定位/视觉/产品线信息？

## 礼盒特殊检查

- [ ] 礼盒系列是否追问了送礼场景？
- [ ] 文案是否适配了用户选择的送礼场景？
