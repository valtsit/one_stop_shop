# 文案风格指南

## 核心调性

**功效卖点型**：直接、有力、易记

## 文案公式

### 公式1：功效+成分
```
[功效词] + [成分A] + [成分B]
```
示例：补气养血，黄芪+当归

### 公式2：人群+场景
```
[适用人群] + [使用场景]
```
示例：熬夜党必备，每天一杯

### 公式3：组合价值
```
[组合名] + [核心卖点]
```
示例：气血双养，内调外养好气色

## 功效词库

### 补气类
- 补气固表、提升元气、增强免疫
- 气血充盈、精力充沛、活力满满

### 补血类
- 补血养颜、红润气色、面色红润
- 气血双养、由内而外、好气色

### 健脾类
- 健脾养胃、脾胃调和、消化顺畅
- 脾胃要养、温和滋补、不上火

### 祛湿类
- 祛湿健脾、湿气拜拜、身体轻盈
- 湿态丢丢、轻松祛湿、无湿一身轻

### 润肺类
- 润肺养阴、清心安神、滋阴润燥
- 补而不燥、温和滋润、不上火

## 人群标签

### 按年龄
- 打工人、加班族、熬夜党
- 中老年人、长辈爸妈

### 按症状
- 气色差、易疲劳、手脚冰凉
- 脾胃虚、湿气重、容易上火

### 按场景
- 日常滋补、换季调理、术后恢复
- 送礼佳品、孝敬长辈

## 禁忌词

### 绝对化用语（禁止）
- 根治、治愈、100%有效
- 最好、第一、唯一
- 包治、神效、特效

### 医疗暗示（禁止）
- 治疗、疗效、处方
- 药用、医用、临床

### 夸大宣传（禁止）
- 立竿见影、一次见效
- 永久、终身、彻底

## 推荐用语

### 安全表达
- 辅助调理、温和滋补
- 日常养生、传统滋补
- 经典配方、古方搭配

### 场景表达
- 每天一杯、坚持饮用
- 由内而外、内外兼修
- 长期调理、循序渐进
