# 电商功效文案生成器 

为电商产品组合生成【XXX】+成分的命名文案。

## 触发条件

用户输入包含以下关键词时触发：
- "写文案"、"功效文案"、"产品文案"、"组合文案"
- 产品名称 + 成分组合（如"黄芪+枸杞"）

## 输入解析

从用户输入中提取：
1. **成分组合**：如"菊花+金银花"、"当归+枸杞"
2. **附加信息**：规格、卖点、适用场景（如有）

## XXX 命名规则

### 核心原则

**描述美好状态，有画面感，有文化感**

### 禁用词（绝对不能用）

#### 功效类（直接描述功效）
- 补气、补血、养血、活血
- 健脾、养胃、消食、化积
- 祛湿、利水、渗湿
- 润肺、养阴、清心
- 明目、安神、固表
- 补肾、滋阴、壮阳
- 清热、解毒、清肝

#### 医疗类（暗示疗效）
- 治疗、疗效、处方、药用
- 医用、临床、康复、痊愈
- 根治、治愈、特效、神效

#### 绝对化用语
- 最好、第一、唯一、100%
- 立竿见影、一次见效
- 永久、终身、彻底

#### 不好的词
- 血色（不好听）
- 常年、日常、每天（太普通）
- 菊银（生硬组合词）

## 核心词提取规则

### 步骤1：分析成分功效
- 当归：补血 → 核心词：红润、血、养
- 枸杞：明目 → 核心词：明、亮
- 菊花：清热 → 核心词：清、凉、爽
- 金银花：解毒 → 核心词：清、凉、爽
- 黄芪：补气 → 核心词：元气、气、力
- 茯苓：祛湿 → 核心词：轻、盈、湿

### 步骤2：找到组合核心效果
- 当归+枸杞：补血明目 → 红润、好气色
- 菊花+金银花：清热解毒 → 清、凉、爽
- 黄芪+当归：补气补血 → 气血、红润
- 黄芪+茯苓：补气祛湿 → 轻盈、元气

### 步骤3：选择核心词
- 选择最能体现组合效果的词
- 选择有画面感的词
- 选择用户喜欢的词

## 生成规则

### 数量

- 每个组合生成 **5-8个** 命名方案

### 核心要求：风格多样，拒绝模板化

**禁止**：每次输出都套用相同的公式结构

**要求**：
- 每个方案都要有不同的表达角度和语言风格
- 混用各种表达方式：诗意、口语、古风、现代、比喻、否定、场景...
- 让人感觉是不同人写的，而不是同一个模板生成的
- 有的雅致，有的直白，有的含蓄，有的活泼

### 质量

- 每个【XXX】控制在 **4-5字**
- 朗朗上口，易于记忆
- 有画面感、文化感、品质感
- 描述美好状态，不直接说功效

### 表达角度参考（灵活运用，不要固定顺序）

| 否定表达 | 说"不"什么 | 不黄不暗、清而不寒 |
| 口语化 | 接地气 | 气血要养、湿气拜拜 |

## 注意事项

1. **绝对不能出现功效词**：如"补气"、"祛湿"、"明目"等
2. **不能暗示疗效**：如"治失眠"、"消积食"等
3. **保持中性**：只描述状态，不承诺效果
4. **有画面感**：用比喻、意象描述美好状态
5. **有文化感**：用"以X润X"等结构
6. **有品质感**：用"双宝"、"双珍"等词
7. **避免生硬组合词**：如"菊银"
8. **避免太普通的词**：如"常年"、"日常"、"每天"

## 格式选择规则

- **用户要求详细/完整分析** → 使用标准输出结构
- **用户快速提问/只要结果** → 使用简洁输出结构
- **默认** → 使用标准输出结构
